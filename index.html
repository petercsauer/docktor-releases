How it works

The Dock becomes a control surface.

docktor doesn't try to replace macOS. It sharpens the one place you already visit all day with previews, widgets, and quick info.

01

Hover

Preview every window for an app before you commit to switching.

02

Glance

Keep music, meetings, weather, a focus timer, and system load beside the Dock.

03

Act

Click a preview, pause a track, check the next event, or clear a timer.

Dock Previews

Find the right window before you switch.

Hover an app and docktor turns the Dock into a live window chooser. Real thumbnails, one-click activation, and no need to open Mission Control.

ScreenCaptureKit thumbnails update in real time
Shows minimized and off-screen windows alongside visible ones
Click to focus the exact window you need

Window Switcher

Cmd+Tab, but you can actually see what you're switching to.

docktor replaces the stock Cmd+Tab switcher with live window thumbnails so you pick the right window on the first try.

Live thumbnails instead of just app icons
See every window, not just the frontmost one
Keyboard-driven — works exactly where muscle memory expects

Privacy

Every prompt has a job.

docktor is most useful when it can see the Dock and render previews. Optional prompts stay scoped to the feature you turn on. No Google OAuth. No account required for core features.
